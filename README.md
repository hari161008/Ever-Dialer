# Ever Dialer

A fast, lightweight Android dialer focused on simplicity and smooth performance.

---

## 📱 Features

• 📞 Clean dial pad for quick and easy calling  
• ⚡ Smooth and responsive performance  
• 📂 View recent calls and manage contacts  
• 🎨 Minimal and modern user interface  
• 🔋 Lightweight app with low resource usage
